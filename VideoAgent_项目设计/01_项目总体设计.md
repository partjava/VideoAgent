# VideoAgent：基于多智能体协作的一键短视频生成系统

## 项目名称

**VideoAgent：基于多智能体协作的一键短视频生成系统**

## 核心流程

```
用户输入一句话
→ DeepSeek 解析任务 & 生成脚本
→ DeepSeek 生成分镜（每段 ≥ 4秒）
→ DeepSeek 精修对白 & 生成提示词
→ Doubao Seedream 生成图片（4路并发）
→ Doubao Seedance 生成视频（3路并发，AI重写防审核拦截）
→ edge-tts 每个分镜独立配音
→ Python 生成字幕
→ MoviePy + FFmpeg 合成视频
→ 质检 & 导出
```

## 技术栈

| 模块 | 技术 |
|------|------|
| LLM | DeepSeek（langchain ChatOpenAI） |
| 图片生成 | 火山引擎方舟 Doubao Seedream 5.0 Lite |
| 视频生成 | 火山引擎方舟 Doubao Seedance 1.5 Pro |
| 配音 | edge-tts（每个分镜独立文件） |
| 编排 | LangGraph StateGraph（`Send` 并行分支） |
| 后端 | FastAPI + BackgroundTasks + MongoDB |
| 前端 | Vue 3 + Element Plus |
| 剪辑 | MoviePy + FFmpeg |

## 模型动态切换

通过 `.env` 文件配置 provider，可随时切换：

- `IMAGE_PROVIDER`: doubao / qwen / comfyui
- `VIDEO_PROVIDER`: doubao / wan / vidu / comfyui
